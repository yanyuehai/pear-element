# 安装

## 版本

pear-element 目前还处于快速开发迭代中。

## 使用包管理器
我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 Element Plus，然后您就可以使用打包工具，例如 Vite 或 webpack。
暂时支持NPM
```
# 选择一个你喜欢的包管理器

# NPM
$ npm install pear-element --save
# YARN
$ yarn add pear-element
```
